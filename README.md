# EcoTrack - Backend

Backend dell'applicazione EcoTrack, una web app di gamification ambientale. Espone le API REST che gestiscono utenti, missioni, punteggi, badge e classifica. Sviluppato in Java con Spring Boot come parte del progetto capstone Full Stack Developer presso Epicode.

## Descrizione

Questo repository contiene la logica server e le API REST di EcoTrack. Gestisce l'autenticazione degli utenti, l'assegnazione delle missioni, il calcolo dei punteggi, lo sblocco dei badge e l'aggiornamento della classifica, con persistenza dei dati su database PostgreSQL.

## Funzionalità

- Autenticazione e gestione utenti
- Gestione delle missioni a tema ambientale
- Calcolo e assegnazione dei punteggi
- Gestione dei badge sbloccabili
- Aggiornamento della classifica in base ai punti
- API REST per la comunicazione con il frontend

## Tecnologie utilizzate

- Java
- Spring Boot
- Spring Data JPA
- PostgreSQL
- API REST

## Architettura

Il progetto è diviso in due repository separati:

- **Backend** (questo repository): API REST in Java + Spring Boot
- **Frontend**: interfaccia utente in React + TypeScript + Vite
